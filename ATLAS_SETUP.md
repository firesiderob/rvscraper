# MongoDB Atlas Quick Setup Guide

I've opened MongoDB Atlas for you in the browser. Follow these steps:

## Step 1: Sign Up
- Use Google/GitHub login for fastest signup
- Or create account with email

## Step 2: Create Free Cluster
- Choose **M0 (Free)** tier
- Select a region close to you
- Click "Create Deployment"

## Step 3: Create Database User
- Username: `admin`
- Password: Create a strong password (save it!)
- Click "Create User"

## Step 4: Get Connection String
1. Click "Connect" button
2. Choose "Connect your application"
3. Copy the connection string
4. It looks like: `mongodb+srv://admin:PASSWORD@cluster0.xxxxx.mongodb.net/`

## Step 5: Paste Connection String Here
Once you have it, paste it in the chat and I'll configure the backend automatically!
